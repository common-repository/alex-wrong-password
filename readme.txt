== Description ==

When someone gets a password wrong or uses the forgotten password form on your site, the administrator is emailed with the details that were entered.
The best way to get fast support is by posting your question in a comment at http://anthony.strangebutfunny.net/my-plugins/wrong-password/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `the directory "wrong-password"` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. When someone gets a password wrong or uses the forgotten password form on your site, the administrator is emailed with the details that were entered.
